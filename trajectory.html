7 轨迹分析

7.1 Diffusion Map

XSeurat2 使用 destiny 软件进行扩散映射（Diffusion Map）分析。分析得到的结果保存在 Seurat 对象中，可以另存为 RDS 文件，或者直接进行可视化作图，包括两种：

  1. 取前两个 Diffusion Component（DC），用类似于 UMAP 和 tSNE 图的方式展示不同组别的细胞
  2. 选择某个 DC，以该 DC 从小到大的方向，用山脊图的方式展示不同组别的细胞的丰度变化

Figure 7.1: 扩散映射（Diffusion Map）

7.2 Monocle2

XSeurat2 使用 Monocle2 软件进行拟时分析（Pseudotime Analysis）分析。

7.2.1 Seurat 对象转 CellDataSet 对象

首先将 Seurat 对象转为 Monocle2 需要的 CellDataSet (CDS) 对象。转换完毕后，XSeurat2 自动用 Monocle2 进行拟时分析，生成的 CDS 对象可以保存为 RDS 文件也可以继续分析。由于拟时分析比较耗时，可以先对细胞进行随机抽样，并增加参与并行计算的 CPU 数目。

Figure 7.2: Seurat对象转 CellDataSet 对象

7.2.2 State 比例与 Root State

拟时分析结束后，我们先看各个 State 中，细胞类型的比例是如何分布的。细胞类型的分布可以帮助我们确定以哪个 State 作为拟时间的 Root。

Figure 7.3: State 比例


默认是 State 1 作为拟时间的 Root，我们可以利用 Reorder Cells 功能，自定义 Root State，重新计算拟时间。

7.2.3 细胞轨迹图

除了拟时间（Pseudotime）之外，我们还可以在细胞轨迹图上用不同的颜色表示各种元数据分类变量的取值，比如细胞类型、State等等。轨迹图还可以拆分为多个子图。

Figure 7.4: 细胞轨迹图

7.2.4 拟时间与基因表达量

基因表达量随拟时间变化的趋势可以用以下几种方法展示：

  1. 用散点图和拟合曲线的方法展示基因表达量随拟时间变化的趋势
  2. 用热图展示基因表达量随拟时间变化的趋势
  3. 在细胞轨迹图上展示基因表达量随拟时间变化的趋势

Figure 7.5: 拟时间与基因表达量-1


除了基因表达量，XSeurat2 还可以用山脊图展示各种细胞类型的丰度随拟时间的变化。和 Diffusion Map 中 DC 的山脊图类似。

Figure 7.6: 拟时间与基因表达量-2

7.2.5 Monocle 结果导入 Seurat 对象

我们可以把 Monocle 生成的结果（例如各个细胞的 Psudotime 和 State 信息添加到 Seurat 对象的元数据中。

Figure 7.7: State 比例

7.3 RNA 速率分析

7.3.1 前期准备

XSeurat2 使用 Velocyto 软件进行 RNA 速率分析。首先需要使用 velocyto 的命令行工具将 CellRanger 输出的 BAM 文件转换为 loom 文件，具体方法见 velocyto 官方文档。

7.3.2 Loom 文件转 Seurat 对象

每个样本的 Loom 文件都准备好之后，可以用 XSeurat2 把 Loom 文件转换为基于 RNA 剪切（Splicing）信息的 Seurat 对象，并保存为 RDS 文件。

Figure 7.8: Loom 文件转 Seurat 对象

7.3.3 合并多个 Seurat 对象

如果有多个样本，可以用合并对象工具，将多个基于 RNA 剪切信息的 Seurat 对象合并为一个 RDS 文件。

7.3.4 把 SpliceAssay 添加到 Seurat 对象

接下来，我们要合并基于基因表达的 Seurat 对象和基于 RNA 剪切的 Seurat 对象，也就是把后者中的 Splice Assay 添加到前者的 Seurat 对象中。这样，在同一个 Seurat 对象中，既有单细胞的基因表达数据，也有 RNA 剪切数据，方便我们进行后续的 RNA 速率分析。

Figure 7.9: 把 SpliceAssay 添加到 Seurat 对象

7.3.5 RNA 速率的计算与可视化

导入前面创建好的 Seurat 对象，并选择所需的元数据分类变量，比如细胞类型（cell_type）。点击 RUN 按钮开始 RNA 速率分析。得到的 RNA 速率结果保存在 Seurat 对象中，并可以在 UMAP 图上展示。

Figure 7.10: RNA 速率的计算与可视化
